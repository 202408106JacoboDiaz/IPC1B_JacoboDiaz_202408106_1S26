## Descripción del Programa
Sistema de gestión de inventario para una tienda de ropa desarrollado en Java con interfaz gráfica Swing. Permite registrar productos, gestionar el stock, registrar ventas y generar reportes en formato HTML.

---

## Funcionalidades
- Agregar productos con código único, nombre, categoría, precio y stock
- Buscar productos por nombre, código o categoría
- Eliminar productos del inventario
- Registrar ventas descontando el stock automáticamente
- Generar reportes de stock y ventas en formato HTML
- Ver datos del estudiante

---

## Instrucciones para ejecutar

### Opción 1 — Desde NetBeans
1. Abrir NetBeans
2. Dirigirse a **File → Open Project** y seleccionar la carpeta del proyecto
3. Presionar **F6** o el botón de *Play* (color verde)

### Opción 2 — Desde el archivo JAR
1. Localizar el archivo `.jar` en la carpeta `dist`
2. Tener instalado Java JDK 11 o superior en el dispositivo
3. Hacer doble clic sobre el archivo `.jar`
